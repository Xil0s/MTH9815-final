## To run the program :
### First generate the data with (compile and run):
```ruby
cd data/
g++ -std=c++11 data_generator.cpp -o data_generator
./data_generator
cd ..
```
### Then compile and run with these commands :
```ruby
 g++ -std=c++11 main.cpp -o main
 ./main
```
